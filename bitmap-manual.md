# **Bitmap.ev3是什么?**

Bitmap.ev3是一个在ev3上实现的地图库，可以在ev3上快速开发带有地图的游戏。

# Bitmap.ev3有哪些函数？

bitmap.ev3很简单，它只有5个函数：

1.**bm_mk**：初始化地图

2.**bm_disp**：显示地图

3.**bm_box**：填充某块区域

4.**bm_del**：取消某块区域的填充

5.**bm_check**：检查某块区域是否被填充
